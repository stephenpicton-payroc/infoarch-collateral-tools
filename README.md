# infoarch-collateral-tools

Reusable collateral, applications, and scripts maintained by the Information Architecture team. Loading collateral files into your AI tool reduces token usage and produces more consistent output — lowering cost and improving quality across AI-assisted tasks.

## What's in this repo

### Tools and applications

Small, self-contained tools for common IA tasks.

| Tool | Description |
|------|-------------|
| [Pandoc Converter](./tools/pandoc-converter/) | Desktop application for converting documents between Word (.docx) and Markdown (.md) format |

### AI collateral

> Coming soon.

Markdown files you can load into an AI tool to provide Payroc style and IA context without re-explaining it each session. Reduces prompt length and token cost for repeated tasks.
